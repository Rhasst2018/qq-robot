# qq-robot
基于CQ的多功能QQ聊天机器人。

### 支持
* 回复at
* 执行JS程序
* 天气预报
* osu!查询
* 黑名单操作  
* 代发消息
* 命令查询
* 机器人接收到的私聊消息自动转发给管理者
* 游戏  
  1. Bulls and Cows猜数字
